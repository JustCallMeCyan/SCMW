# SCMW (Stop Closing My Window)

A Fabric mod for Minecraft 1.21.8 and 1.21.10 that prevents menus from being forcibly closed by game events.

## Features

- Keeps menus open during teleportation
- Prevents dimension changes from closing menus
- Allows menus to open even in nether portals
- User can still close menus normally with ESC or close button

## Requirements

- Minecraft 1.21.8
- Fabric Loader
- Fabric API
- Java 21

## Installation

1. Install Fabric Loader for Minecraft 1.21.8 or 1.21.10
2. Download Fabric API
3. Place the SCMW mod JAR in your mods folder
4. Launch Minecraft
